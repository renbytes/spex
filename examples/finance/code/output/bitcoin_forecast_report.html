Executive Summary

This report presents a time series forecast for the daily price of Bitcoin (BTC) for the next 365 days. The analysis is based on historical price data, utilizing a SARIMA (Seasonal AutoRegressive Integrated Moving Average) model to generate the forecast. The model parameters used were SARIMA(1, 1, 1) with a seasonal component of (1, 1, 1, 12). A 90-day moving average was also calculated to smooth out price volatility and identify trends.
